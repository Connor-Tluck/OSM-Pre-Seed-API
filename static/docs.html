API Information

Available Endpoints:

  • POST /query - Get raw OSM data
  • POST /generate - Generate files (reports, plots, maps)
  • POST /csv-rollup - Generate CSV rollup with feature counts
  • GET /feature-types - List available feature types
  • GET /mach9-feature-types - List Mach9 engineering feature types
  • GET /examples - Get example requests

Rate Limits:

  • Query: 20 requests/minute
  • Generate: 10 requests/minute
  • Download: 30 requests/minute
